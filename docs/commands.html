BOT COMMANDS

API endpoints here

Bot prefix: !!

(arg) means the argument is required

[arg] means the argument is optional

MODERATION COMMANDS

ban (player) | Bans a player

unban (player) | Unbans a player

bans | Gets the current list of banned players

kick (player) | Kicks a player


UTILITY COMMANDS

ping | Returns the ping of the bot

ping api | Returns the ping of the of the website's API
