Bot prefix: !!

(arg) means the argument is required

[arg] means the argument is optional

MODERATION COMMANDS

ban (player) [time] | Bans a player for an optional amount of time (none is permanent)

unban (player) | Unbans a player

bans [date] | Gets the current list of banned players from an optional date

kick (player) | Kicks a player

kickothers | Kicks all players except for moderators or higher

kickall | Kicks all players in the server besides you


UTILITY COMMANDS

info server (server) | Return the info of a server

ping | Returns the ping of the bot and the API's
